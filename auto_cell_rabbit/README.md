# 🐰 守护小兔子 - 元胞自动机游戏

基于康威生命游戏（Conway's Game of Life）的教育性互动游戏。

## 🎮 游戏模式

### 👀 观察模式
- **初始状态：** 屏幕中央有一只精心设计的像素兔子
- **玩法：** 点击"开始演化"，观察兔子在康威规则下会变成什么样子
- **教育目的：** 让小朋友看到"精心设计的图案在简单规则下也会产生意想不到的变化"

### 🛡️ 守护模式
- **挑战目标：** 在兔子周围画细胞，尝试保护兔子不被破坏
- **玩法：** 
  1. 点击网格可以画/擦细胞
  2. 设计好防护布局后，点击"开始演化"
  3. 观察你的布局能否守护兔子存活
- **成功标准：** 兔子区域存活细胞数超过 30%

## ✨ 特性

- **自由绘制的初始兔子** - 不受康威规则限制，纯视觉设计，真实可爱
- **标准康威规则演化** - 游戏启动后严格按照经典规则运行
- **多种族支持** - 可选启用多种族模式，观察种族竞争
- **实时统计** - 显示代数、兔子存活细胞数、种族分布

## 🎯 康威生命游戏规则

1. **活细胞**：
   - 周围有 2-3 个邻居 → 继续存活
   - 邻居太少（<2）或太多（>3）→ 死亡

2. **死细胞**：
   - 周围恰好有 3 个邻居 → 复活

## 🎨 设计说明

**初始兔子图案：**
- 参考用户提供的像素兔子图片设计
- 包含：长耳朵、圆头、眼睛、鼻子、嘴巴、身体、四条腿
- 约 150+ 个细胞组成
- **不是 Still Life** - 演化后会改变

**与其他版本的区别：**
- `auto_cell_single` - 经典康威游戏，随机或手动绘制
- `auto_cell_multi` - 多种族竞争模式
- `auto_cell_rabbit` - **专为小朋友设计的教育游戏，有可爱主题和挑战模式**
